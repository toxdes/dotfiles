# Sway Configuration

## Keybindings

Modifier key: <kbd>Super</kbd> (Windows key)

### Basics

| Keybinding | Description |
|-----------|-------------|
| <kbd>Super</kbd>+<kbd>Return</kbd> | Open terminal |
| <kbd>Super</kbd>+<kbd>Shift</kbd>+<kbd>q</kbd> | Kill focused window |
| <kbd>Super</kbd>+<kbd>d</kbd> | Open application launcher |
| <kbd>Super</kbd>+<kbd>Shift</kbd>+<kbd>c</kbd> | Reload configuration |
| <kbd>Super</kbd>+<kbd>Shift</kbd>+<kbd>e</kbd> | Exit Sway |
