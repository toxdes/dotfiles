# Neovim Configuration

## Keybindings

Leader key: <kbd>Space</kbd>

### Telescope (Fuzzy Finder)

| Keybinding | Description |
|-----------|-------------|
| <kbd>Space</kbd>+<kbd>f</kbd> | Find files |
| <kbd>Space</kbd>+<kbd>b</kbd> | Find files |
| <kbd>Space</kbd>+<kbd>r</kbd> | Live grep search |
| <kbd>Space</kbd>+<kbd>c</kbd>+<kbd>h</kbd> | Help tags |
| <kbd>Space</kbd>+<kbd>c</kbd>+<kbd>m</kbd> | Man pages |
| <kbd>Space</kbd>+<kbd>c</kbd>+<kbd>c</kbd> | Colorschemes |

### Editor

| Keybinding | Description |
|-----------|-------------|
| <kbd>Space</kbd>+<kbd>=</kbd> | Indent entire file |
| <kbd>Ctrl</kbd>+<kbd>c</kbd> | Clear search highlight |
| <kbd>Ctrl</kbd>+<kbd>d</kbd> | Scroll down with centered cursor |
| <kbd>Ctrl</kbd>+<kbd>u</kbd> | Scroll up with centered cursor |
| <kbd>x</kbd> | Delete character without copying |

### NvimTree (File Explorer)

| Keybinding | Description |
|-----------|-------------|
| <kbd>Space</kbd>+<kbd>t</kbd> | Toggle file tree |

### Configuration

| Keybinding | Description |
|-----------|-------------|
| <kbd>Space</kbd>+<kbd>c</kbd>+<kbd>r</kbd> | Save and source current file |
| <kbd>Space</kbd>+<kbd>c</kbd>+<kbd>c</kbd> | Edit init.lua |

### Motion (Visual Mode)

| Keybinding | Description |
|-----------|-------------|
| <kbd>J</kbd> | Move line down |
| <kbd>K</kbd> | Move line up |
